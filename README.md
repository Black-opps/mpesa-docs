## Platform Overview

Production-grade SaaS platform for M-PESA transaction analytics with:
- ✅ API Gateway (CORS, rate limiting, circuit breaker)
- ✅ Authentication & Authorization (JWT)
- ✅ Multi-tenant analytics
- ✅ React Dashboard
